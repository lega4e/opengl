# Hello

Первая программа с использованием OpenGL (не без помощи GLFW).
Писал по этой [https://ravesli.com/urok-3-nash-pervyj-proekt-v-opengl-sozdanie-okna/](статье).

Для сборки и запуска требуется установленные библиотеки `GLFW` и `glad`.

Приложение способно только отрисовывать экран тремя разными цветами:
тёмно-синим, тёмно-зелёным и тёмно-красным.

Управление:

- `R` — отрисовывать экран тёмно-красным
- `G` — отрисовывать экран тёмно-зелёным
- `B` — отрисовывать экран тёмно-синим
- `Escape` — завершить программу
